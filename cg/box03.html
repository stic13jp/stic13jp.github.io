[ Comment  ] ゆりちゃんに見え……ないですよね
---------------------------------------------------------------
[ Description ]

　タレ目っ娘(コ)を描いてみたかったんですが、うまくいきませんでしたね。ウェディングピーチは観ていなかったんですが、もうすぐ終わるそうなので観始めました。いや、ピーチは関係ないんですけど(^^;。柳葉×ようすけ、なのか～(ばき)。ちなみに、このCGは、いったん初号機(Aptiva 755)に持って行って、Paint Shop ProでGIFに変換しました。

　最初100dpiで読み込んだところ、お口がムスっとしてしまったので200dpiで読み込み直しました。おかげででかくなってしまいました。でも、ファイルサイズはでかくないから許してください(^^;。いつも首を太く描いてしまうので、細くなるように気をつけました。また、耳が出るのもおかしいので髪で隠しました(下書きの時点では出ていた)。

　ニューマシンでPhotoShopを利用して、背景を処理。正確には、色域指定の自動選択で線画のみ線画レイヤーにコピー＆ペースト。GIF89aのインターレースで、出力しました。PSDファイルが残っているので、気が向いたら色塗りするかも……多分、そんなヒマはないと思うけど(T_T)。 

[ History  ]

　1996/03/18　描いて、スキャナで読み込んで～。
　1998/08/24　修正。

---------------------------------------------------------------
[ Description ]

　というわけで、全身像にチャレンジ。以前に描いた全身像のファイルが、どこかにいってしまいました。ま、探せば出てくるでしょうし、南風から、もう一度落とせばいいだけの話なんですが。某イラストを元に全身像を描いてみました。ポーズは変えてあります。手を描くのは無理そうだったので、後ろにまわしてもらいました。

　腰まわりが、ぼてっとしちゃっていますね。もっと、きゅっとしまっていた方が格好いいですよね。あと服のしわも、いまひとつ。これわ、はっきりいってどう描いていいのかわからなかったので、いい加減です。胸の線も、わかりませんでした。あんまり大きく描いて、えっちっぽくしたくなかったし。脚のバランスも、おかしいかもしれません……って、おかしなところばっかりですね(^^;、(;_;。

　そうか。ヒカリちゃんにしては、髪が長すぎるんですね……って、ヒカリちゃんぢゃないんですが(をひこら)。チャイナなパジャマ姿がよかったですよねえ。いや、そんなことわどうでもいいんですが、全身像を描くときは服がポイントになるわけで、服って難しいなあということがいいたかったんです(^^;。服を着させてあげていないCGも多いですけどね。

　で、何かおすすめのデッサン本というか、イラストの描き方本みたいなものがありましたら、よろしければ教えていただけませんでしょうか。安いやつがいいです(をひ)。本自体も大きくなく、厚くないやつがいいです(をひ)。それから、それから……いい加減にしろ>をれ。あと、GIFにしたらヒカリちゃんとわ別人になってしまいました←だから、ヒカリちゃんぢゃないんだってば。
