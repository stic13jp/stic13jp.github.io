Old Cathy's Profile #2

[ Description ]

　2006/08、まわりからすれば「いまごろかよ」とでも言われそうなタイミングで、個人的には待望のAMD Athlon(64 X2)にてPCを組みました。しばらくPCが不安定で困っていましたが、Palm Desktopが原因と判明。日本語版ver4.01ではなくver4.1.4E(日本語化Patch)で、とりあへずOKそう……と思ったら、元に戻ってガックリ。2007/04、メモリをさし直してもNG。2007/05、メモリを交換したらOKそう。2008/08、BIOSアップデートが原因なのかわからないけど、電源とM/Bが昇天。たまにブルースクリーンが発生していたのも、結局なにが原因かわからず仕舞いでした。AMDプラットフォームには、がっかりだよ! ←関係ないと思う。
