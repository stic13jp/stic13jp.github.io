Old EVA-03's Profile #1

[ Description ]

　2000/09/08から2006/05/13まで就役していた、旧参号機(Celeron 466MHz)です。いわゆる自作機なので、型番ではなくCPUにて区別します(CPUだけパーツを交換するということは、自分はしないので)ハードウェアが原因なのかOS(Windows 98 SE)が原因なのか、それとも単にネットワーク(WAN)が遅いだけだったのか(PIAFS 32K)、とにかくパフォーマンス改善のため組み直すことにしました。ケース(ただし電源は交換)やFDD・DVD-Rドライブなどは流用したため、以下の構成で残っているマシンはありません。6年近くも、よくもまあがんばってくれたものです。と書きつつ、初号機(EVA-01)は1998年の夏からずっと現役ですががが。
