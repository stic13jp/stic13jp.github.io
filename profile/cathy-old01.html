Old Cathy's Profile #1

[ Description ]

　2000/12/11、Celeron 466MHzのPCを実家に持って行ってしまったため、Pentium III 800MHzで自宅用のPCを組み直しました。カネがないので、FSB 133MHzではなくFSB 100MHzのシステム(CPUとメモリ)としました。当初メモリ容量は128MBだったのですが、のちに128MB増設して合計256MBとしました。2001/08/19、CanonのプリンターBJ F870を購入。2002/06/29、CanopusのキャプチャーカードMTV1000を導入(譲渡)。2002/10/25、PDAのPalm(SONY CLIE)を購入しNotesのカレンダーと同期していました(過去形)。2004/09/25、HDD(40GB)がクラッシュ。その後、増設していたデータ用のドライブ(120GB)もクラッシュしたため、こちらも換装しました。

　2004/09/26、環境を再構築。2005/04/23、iPod Photoを購入。のちに動画再生iPod(第五世代iPod)へ移行し、iPod Photoは実家へ。2005/09/04、無線LANルーターを購入しましたがWAN回線はフレッツISDNのまま。そのため2006年になってから、Podcastの取得に時間がかかることや、そもそもPCのスペック不足を感じることが多くなってきたため、2006年夏にAthlon 64 X2にて組み直しました。すでにCore 2 Duoが出ていた時期ではありましたが、前々から「次はAthlonで」と思っていたため初志貫徹(のつもり)。いまではPCのあまりの不安定ぶりに、Core 2 Duoで別に組み直したいぐらい←2007/04、本当にCore 2 Duoで別に一式組み直しました。
