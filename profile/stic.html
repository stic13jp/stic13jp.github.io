[ Description ]

　実家のハードウェア構成は、以下のようになっています^Hした。

  • PC-9821Xa16/W16改　　　　零号機 (EVA-00: PROTOTYPE)
  • 自作PC/AT互換機 　　　　　　初号機 (EVA-01: TEST TYPE)
  • VAIO type T (VGN-TZ90NS)　弐号機 (EVA-02: PRODUCTION TYPE)
  • 自作PC/AT互換機 　　　　　　参号機 (EVA-03: BARDIEL)

　零号機(EVA-00)は、1997/01/18にPC-9801FS2改からPC-9821Xa16/W16へ換装し、のちにEVA-00専用主記憶装置を増設。そして、HDDがクラッシュ(;_;)。その後、Windows NT ServerでPDC兼DHCPサーバー兼WINSサーバー(兼Notesサーバー)を担う。1998/03/09、弐号機(EVA-02)が再就役。EVA-02専用主記憶装置を増設し、外付CD-ROMドライブ・ポートレプリケーター等のB型装備も準備が完了。1998/07/18、初号機(EVA-01)が自作PC/AT互換機として覚醒(以前の儀体は参号機へ)。いまとなってはたいしたことのないPentium II 400MHzですが、メモリ使用率はスカスカです。CPU使用率はSETI@HomeとUD Agentで100％でしたが、UD AgentがBOINCに移行してからは休憩中です。

　2000/09/09、参号機(EVA-03)がAptiva 755改から自作PC/AT互換機に乗っ取られました。ここに「mailください」って書いていたら、「以前、メル友募集してましたよね」とかゆーヘンなメールが来ちゃったよ(ぢつわ、有名な平田メールだった模様)。Talk Room(≠Chat)でも可……ってゆー記述も、荒らしの原因になってしまうのだろうか。イヤな世の中になっちまったもんだよ。しくしく(T_T)。2001/12/02、不調につき弐号機の儀体(VAIO NOTE 505X)を新しいもの(VAIO NOTE 505V/CBP)と交換しました。2006/01/22、弐号機の調子が悪いため、VAIO NOTE 505V/CBPのHDDを換装。そういえば上には「実家の」って書いてありますけど、弐号機は普段自宅にあります。

　2006/05/13、さすがに6年近くも使っていると古くなってきて、処理が重たいので参号機の諸々パーツを交換(旧参号機その壱)。OSも、Windows XP Professionalに変更しました。2006/08/27、自宅のPC(Cathy)も参号機同様に重くなってしまい、普段の利用にストレスを強く感じるようになってしまったのでパーツを交換(旧Cathyその壱)。ところが突然フリーズしたりキーボード/マウスを動かしても無反応の状態になってしまうことが、いつまでたっても解決しない。2007/04/14、Core 2 Duoで別に一式組み直しました。2007/05/05、Cathyのメモリを交換したところ不調はピッタリとまりました(なんてわけにはいかなくて、結局最後まで不安定なままでした)。

　2007/07/27、VAIO type T(OWNER MADE MODEL)を購入。2008/08/30、CathyをAMDプラットフォームからIntelプラットフォームに戻しました(旧Cathyその弐)。2009/12/25、NAS(2TB)の空き容量が足りなくなってきたので4TBのものに更改。2010/04、実家のプリンター(Canon BJ F900)が壊れたので複合機(Canon MP560)を購入。2011/07、実家の無線LANルーターを交換(Aterm WR8700N)して2TBのNASを回収。2014/04、参号機の換装とあわせて零号機を廃棄(旧参号機その弐)。2020/08/08、実家のプリンターと無線LANルーターを交換。
