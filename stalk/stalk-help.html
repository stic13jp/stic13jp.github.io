STalk 2.5 Help

This talk room is not chat room but one line BBS.

JavaScriptが使える場合、以下の機能が有効になります。

  • 部屋を選択しただけで、その部屋に移動する(Goボタンを押す必要がない)。
  • テキスト・ボックスに、自動的にフォーカスがうつる。

テキスト・ボックスに入力できるコマンドは、以下の通りです。 注: /uは使えません。;-)

/r
Reload: 読みなおし
/r n
Reload n: n行前から読みなおし
/ra
Reload All: 最初から読みなおし

テキスト・ボックスに何も入力しないで Sendボタンを押すことによっても、Reloadできます。
