タイムゾーンを変更する方法

CentOS 6編

　タイムゾーンに関する設定ファイルを指定場所にコピーする、という考え方みたいです。rootになって、以下のコマンドを実行します。 

# cp /usr/share/zoneinfo/Japan /etc/localtime

　確認方法は、以下のコマンドを実行します 

# date

　JSTになっていたら、OKです。「CentOS 6 のタイムゾーンを変更する手順 | WEB ARCH LABO」を参考にさせてもらいました。

CentOS 7編

　コマンドが変更になりました。 

# timedatectl set-timezone Asia/Tokyo

　確認方法は、以下のコマンドを実行します 

# timedatectl

　JSTになっていたら、OKです。「CentOS 7 : システムのタイムゾーンを設定する ： Server World」を参考にさせてもらいました。
