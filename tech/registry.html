レジストリの掃除

　ここでいう「掃除」とは「再構成」を意味します。そもそもレジストリとはなにか/どうしてレジストリを掃除するのか等々は、他に詳しく説明しているサイトがあるので、そちらを参照してください。またレジストリ関連ツールのリスト/紹介も、ここでは行ないません。

Windows 95/98

　Windows 95/98のレジストリについては、下記ソフトウェアの存在する「Regist"o"ry Rev.2」のサイトが詳しいです(が、リンク先ドメインが消失してしまいました)。自分はツール派なので、レジストリを一旦書き出してから再構築……なんてことはせず、以下を利用します。

Registr!p

　Registr!pは直接レジストリの掃除を行なうツールではなく、レジストリの中身を確認するためのツールです。修復や再構成の機能はないので別のツールが必要になりますが、RegConでエラーが出た場合などにRegistr!pが必要になるときがあります。Windows 95/98専用です。

RegCon

　実際にレジストリの再構成を行なうのは、こちらのRegConになります。こちらも、Windows 95/98専用です。両ソフトウェアの「窓の杜」での紹介は「鹿山雅志のオンラインソフトつっこみレビュー」(リンク切れ)です。

　手順は、基本的にRegConを起動して「OK」をクリックするだけ。バックアップを圧縮しなくてもよいのであれば、オプションの「◆バックアップ」で「そのまま」を選択します。

Windows 2000/XP

　あちこち見て「RegSeeker」や「EasyCleaner」などを試してみましたが、自分としては以下が合っているようです。

  1. RegCleanerで、不要エントリーを削除
  2. NTREGOPTで、レジストリを再構築(圧縮)

　どちらのツールも日本語化パッチがあります。せっかくなので適用しておきます。不要エントリーは、RegCleanerで「N/A」になっているものをガシガシ削除してしまいます。同じくRegCleanerで「ツール」→「レジストリクリーンアップ」→「それらを全て行う」を実行します。次に、NTREGOPTを起動します。こちらは起動すれば処理が自動的に開始されますので、処理が完了したらOS再起動しておしまいです。

　またOSは限定されますが、上記「Regist"o"ry」のサイトにはWindows XPのレジストリ再構成について記述があります(が、リンク先ドメインが消失してしまいました)。
